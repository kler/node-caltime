(inner) timeZone(inTZ) → {object}

Functional constructor which creates an instance of a TimeZone. Each TimeZone object is immutable.
Parameters:
Name Type Description
inTZ string String identifying the timezone. Valid timezone strings are valid TZ environment variable values. See https://www.iana.org/time-zones for more details. An error is thrown if the string is not specified or is not one of the permitted TZ values.
